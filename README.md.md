# GeoMapper Customer Information - README

This repository contains the following: 

- an instruction manual for setting up the GeoMapper OEM software
- reference CAD models for a handheld version of GeoMapper
- reference design for a webserver to interact with GeoMapper

Please note that the reference designs are provided as is, and are untested for the purposes of your setup. They should not be copied directly with extensive engineering to ensure compatibility and functionality.
